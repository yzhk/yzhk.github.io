程序设计基础总结（一）

程序设计基础总结（一）

数据的输入与输出

1.输入

输入函数

scanf("%格式控制符"，地址列表)；

scanf()函数中的格式控制符（常用）

d：读入int类型的整数（十进制）

f：读入float类型的数（十进制）

c：读入一个字符（输入时，空格和转义字符均为有效字符）

s：读入一个字符或字符串，遇空格、制表符或回车键结束（与数组一起用）

scanf的格式修饰符（放在%和格式控制符之间）

h：用于d前，指定输入为short型整数

l：用于d前，指定输入为long型整数；用于f前，指定输入double型实数

m：任意正整数，指定输入数据长度，遇空格或不可转换字符结束

*：抑制符，通常与m修饰符一起用，表示不输入m个长度的数据

输入注意点

1.数据输入时，当格式控制序列中存在非格式字符时，输入时也要一起输入。如：

scanf("a=%d,b=%d:c=%d",&a,&b,&c); 应输入a=1,b=2:c=3<再回车>(""中的非格式字符是什么输入法，输入时就要用什么输入法)

2.输入数据时遇到以下情况则认为该数据输入结束：①遇空格键、Tab键或回车键；②有指定数据长度时，自动按长度截取；③遇到非法输入。

实例一

int a,b;scanf("%3d%2d",&a,&b);printf("a=%d b=%d",a,b);

1)输入：12345678；输出：a=123 b=45；说明：按指定长度截取数据，即情况②。

2)输入：1<空格>234；输出：a=1 b=23；说明：遇到合法间隔符即①，认为a变量的输入结束，再按指定长度截取数据给b，即②。

3)输入：12，34；输出：a=12 b=-858993460；说明：遇到非法输入“，”输入结束,即③，将12赋值给a，b的值则是位置的。

实例二

int a,b;scanf("%3d%*2d%2d",&a,&b);printf("a=%d b=%d",a,b);

输入：123456789；输出：a=123,b=67;说明：%*2d表示不输入2个长度的int数据，即输入的45不接收。

2.输出

输出函数

printf("%格式控制符"，输入列表)

printf()函数中的格式控制符（常用）

d：输出int类型的整数（十进制）

f：输出float或double类型的数（十进制）

c：输出一个字符（输入时，空格和转义字符均为有效字符）

s：直接输出字符串，可无。例如printf("%s","ABC");输出：ABC

printf()函数的修饰符

m：任意整数，数据长度<|m|，若m>0，在数据左补空格，若m<0，则在数据右补空格；否则按实际输出

.n：①对实数，确定有几位小数；②对字符串，指定最终输出的长度③对整数，n<=m时，在数的左边补0，且0的个数与原整数的位数的和为n，剩余的在输出的数的左边补空格

0m：在输出数据左边补0，且0的个数与数据的长度的和为m

l：用于d前，指定输出精度为long型；用于f前，指定输出精度为double型

注

putchar(ch);相当于printf("%c",ch);

getchar()：输入一串字符，第几次给ch赋值，就把字符串中的第几个赋值给ch

运算符与表达式

1.在int型数参与的运算中，如果实际计算的结果有小数部分，则输出的结果为舍去小数的整型数，如“15/4”的结果为3而不是3.75，“1.2+1.7”的结果为2而不是2.9。利用这个性质可以求四舍五入后的整数，即在整个计算式后面加0.5，如“（15/4）+0.5”“（1.2+1.7）+0.5”。

2.++i为先加1，再使用，i++为先使用，再加1

类金字塔型题

#类金字塔型题
###解法：把复杂的整体分解成几个部分（包括空白部分），再分别用循环语句实现（一般有几个部分就有几个for语句）。
##1.输出：
正输：
0
00
000
0000

反输：
0000
000
00
0
语句1（正输）：
for(i=1;i<=n;i=i+1) //输出i列(基本没什么变化)
{
for(j=1;j<=n;j=j+1) //输出j个(个数变化取决于两个’ ; ‘之间的表达式)
{
printf(“0/%d/%c”[, ]) //输出0，输出字符取决于“ ”和[ ]之间的表达式
}
}

  • tips1:输入正反（左右）
    printf中[]的表达式：正输：a+j ；反输：a-j。
    其中a为常数,可从题目中求出，且以上表达式可能会用到' i '
  • tips2:输入正反（上下）
    第二个for语句中两个' ; '的表达式：正输：a*i+b ；反输：a*n-b*i+c。
    其中a，b，c为常数，可从题目求出

##2.输出：
正输:
0001
00111
011111
1111111

反输：
1111111
011111
00111
0001
###分解成0三角和1三角
语句2（0三角反输，1三角正输）：
for(i=1;i<=n;i=i+1) //输出i列(基本没什么变化)
{
for(j=1;j<=n-i-1;j=j+1) //输出0三角
{
printf(“0”)
}
for(j=1;j<=2*i-1;j=j+1) //输出1三角
{
printf(“1”)
}
}

##3.输出：
正输:
0001222
0011122
0111112
1111111

反输：
1111111
0111112
0011122
0001222
###分解成0三角，1三角和2三角
语句3（0三角反输，1三角正输，2三角反输）：
for(i=1;i<=n;i=i+1) //输出i列(基本没什么变化)
{
for(j=1;j<=n-i-1;j=j+1) //输出0三角
{
printf(“0”)
}
for(j=1;j<=2*i-1;j=j+1) //输出1三角
{
printf(“1”)
}
for(j=1;j<=n-i-1;j=j+1) //输出2三角
{
printf(“2”)
}
}

用scanf输入字符时字符无效的解决办法

关于用scanf输入字符时字符无效的解决办法

问题

用scanf输入字符串，运行后出现提示:

0x796B98F1 (ucrtbased.dll)处(位于 1.exe 中)引发的异常:

0xC0000005: 写入位置 0x007D1343 时发生访问冲突。

以及下方提示：

<字符串中的字符无效>

错误的地方:

#include <stdio.h>

int main()

{

char a, b, c;

scanf_s(“%c %c %c”, &a, &b, &c);<<<<<<(错误)

printf(“%d,%d,%d”, a,b,c);

return 0;

}

解决方法

可在scanf中输入的字符后输入“整数，”，如下：
#include <stdio.h>

int main()

{

char a, b, c;

scanf_s(“%c %c %c”, &a, 1，&b, 1，&c，1);

printf(“%d,%d,%d”, a,b,c);

return 0;

}
