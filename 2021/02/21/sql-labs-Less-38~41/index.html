小结

1.堆叠注入
注入时，用‘;’把多个语句分开，这样注入时就可以同时运行多个语句。和union操作符不同的是，union使用的语句类型是有限的。
详细：https://www.cnblogs.com/lcamry/p/5762905.html

2.增删查改语句

1)增：
增数据：insert into 表名（例1，例2，…….）values (v1,v2,…….);
// insert into student(id,name,grade) values (1,’zhangshan’,98);
增数据库：create database 数据库名;
增除表：create table 表名;
2)删：
删数据:
delete from 表名 [where id=1];
//delete from users where id=1;
删数据库：drop database 数据库名;
删除表：drop table 表名;
删除表中的列:alter table 表名 drop column 列名;
3)改：update 表名 set 字段=’值’ [where 条件]；
// update users set username= ‘ABC’ where id=’1’;
4)查：select */字段列表 from 表名 [where 条件];
// select */username from users [where id=’1’];
